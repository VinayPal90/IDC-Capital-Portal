# 🏦 IDC Capital Portal – Full Stack Assessment Project

IDC Capital Portal is a full-stack web application developed as part of the **IDC India Full Stack Intern Assessment**. The application demonstrates the **Collateral-as-a-Service** model by providing an interactive platform where users can understand, calculate, and visualize different aspects of the lending ecosystem.

The project focuses on building a clean, scalable, and user-friendly application within the given 48-hour assessment timeline while showcasing frontend, backend, API integration, and database management skills.

🌐 **Frontend Live:** https://idc-capital-portal-gg84jit9g-vinaypal90s-projects.vercel.app/

🌐 **Backend API:** https://idc-capital-portal.onrender.com

---

# 🚀 Features

- 💰 Deal Calculator for loan distribution
- 📊 Interactive dashboard
- 🏦 Property Owner, Bank/NBFC & Startup views
- 📈 Financial breakdown with visual representation
- ⚡ REST API based architecture
- 🔄 Frontend & Backend integration
- 📱 Fully Responsive Design
- 🎨 Modern UI built with Tailwind CSS
- 🔒 Environment variable support
- 🚀 Production deployment (Vercel + Render)

# ⚙️ How to Run Locally

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/idc-capital-portal.git
```

---

## 2️⃣ Install Dependencies

### Backend

```bash
cd server
npm install
```

### Frontend

```bash
cd client
npm install
```

---

## 3️⃣ Environment Variables

This project includes a **`.env.example`** file.

Copy it and create your own `.env` file.

Example:

```env
PORT=5000

```

> Never commit your actual `.env` file. Only use `.env.example` as a reference.

---

## 4️⃣ Run the Project

### Backend

```bash
npm run dev
```

### Frontend

```bash
npm run dev
```
